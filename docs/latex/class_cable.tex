\doxysubsection{Detailed Description}
Represents a cable with specific attributes and devices connected to it. 

The {\ttfamily \doxylink{class_cable}{Cable}} class encapsulates information about a cable including its type, system usage, input/output characteristics, and the devices it connects to.
